# Лабораторная работа №4 Архитектура сети

## Части

Лабораторная работа разбита на 3 части:
1) Создание сети в соответствии с топологией и настройка основных параметров устройств
2) Настройка и проверка 2 пулов DHCPv4 сервера на маршрутизаторе R1
3) Настройка и проверка ретрансляции DHCPv4 на маршрутизаторе R2
