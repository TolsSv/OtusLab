# Лабораторная работа №8 EIGRP

#### Цель: 

Настроить EIGRP в Санкт-Петербурге используя именованный EIGRP.

При этом необходимо учесть:
- R32 получает только маршрут по умолчанию;
- R16-R17 анонсируют только суммарные префиксы.
